# react-native-device-calendar

A modern React Native module to create and manage calendar events using native Android and iOS APIs.

## Installation

```sh
npm install react-native-device-calendar
```

## Usage


```js
import { multiply } from 'react-native-device-calendar';

// ...

const result = await multiply(3, 7);
```
